# Maintenance-Management-System

A maintenance management system to track company assets, manage maintenance requests, and assign repair tasks to specialized teams.

When an asset such as a machine or computer requires maintenance, a request can be created and assigned to a relevant team. 

The system allows tracking the request status from creation to completion, ensuring better visibility, accountability, and faster resolution.
